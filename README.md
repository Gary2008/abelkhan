# ablekhan
abelkhan是一个轻量级的游戏服务器框架，基于c#开发，使用mongodb提供数据存储，目标是提供一个稳定、高效、可扩展的服务器框架。  

## 架构
* center 中心服务器
* dbproxy 数据库代理
* gate 网关服务器
* hub 微服务节点
